# ibca-wp-theme

## このリポジトリは何？
- 役割：IBCAサイト（WordPress / TCD）の「コードだけ」を管理する場所
- 目的：子テーマ・CSS・functions などをGitHubで履歴管理し、戻せる状態にする
- 状態：運用中

## ここに入れる（OK）
- 子テーマ一式（/wp-content/themes/xxxx-child/）
- functions.php
- 追加CSS（テーマのカスタムCSS含む）
- 固定ページに貼るHTMLブロック（CTAなど）
- テンプレートの調整（必要な範囲）

## ここに入れない（NG）
- 投稿記事本文（WordPress側で管理）
- 管理画面の設定値（基本はWP側）
- メディア画像（必要なら assets リポジトリへ）

## 次にやること（最初の1歩）
- [ ] まず「子テーマ」を作る（または現状の子テーマを確認する）
- [ ] 変更したファイルだけをこのリポジトリに入れていく
